# Obsidian Migration

파편화된 지식을 Obsidian으로 통합해주는 AI 마이그레이션 솔루션

---

## 1. 누구의 어떤 문제를 푸는가

Notion, Slack, Google Docs 등 여러 툴을 동시에 사용하는 지식 노동자들은 두 가지 문제를 겪는다.

첫째, 지식이 분산되어 있어 정작 필요한 순간에 활용하기 어렵다. 어디에 적었는지 기억나지 않아 여러 툴을 뒤지다 시간을 허비한다. 

둘째, Obsidian이 최고의 지식 관리 도구라고 들어봤지만 시작하지 못한다. 지금까지 쌓아온 수백 개의 문서를 구조에 맞게 이전하는 작업이 너무 방대하기 때문이다.

분산된 구조로 쌓인 지식은 검색도, 연결도, 활용도 어렵다. 도구가 많을수록 오히려 지식은 더 깊이 묻혀버린다.

---

## 2. 핵심 동작

**한 번의 마이그레이션으로 지식 구조를 잡고, 이후엔 새로운 정보가 자동으로 쌓인다.**

**Step 1 — 마이그레이션**
1. Notion, Slack, Google에 흩어진 데이터를 연결하고 Obsidian으로 옮길 대상을 선택한다
2. AI가 데이터를 통합 분석해 폴더 구조, 태그 체계, 노트 간 연결 관계를 자동으로 설계한다
3. 설계된 구조 그대로 Obsidian에 저장된다. 처음부터 체계적으로 연결된 지식 베이스가 완성된다

**Step 2 — 지속 수집**

마이그레이션 이후, 새로운 Slack 메시지·Notion 페이지·Google Docs가 생길 때마다 AI가 분석해 Obsidian에 최적화된 형태로 자동 누적한다. 한 번 만들어진 구조 위에 지식이 매일 성장한다.

---

## 3. 완성되면 누구에게 보여줄 것인가

Obsidian에 관심은 있지만 마이그레이션 장벽에 막혀 시작하지 못한 사람들. 구체적으로는 다음과 같은 특징을 가진 잠재 고객이다.

- 업무에서 3개 이상의 툴을 동시에 사용하는 직장인·프리랜서
- Obsidian의 존재를 알고 있고, 써보고 싶다는 생각을 한 번이라도 해본 사람
- 과거에 쌓아온 문서와 메모가 많아 새로운 툴로의 전환을 망설이는 사람

이들에게 "지금 당장 시작할 수 있다"는 것을 보여주는 것이 목표다.
